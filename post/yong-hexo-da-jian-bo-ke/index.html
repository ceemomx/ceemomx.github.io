用hexo搭建博客

2017-11-29 3 min read # nodejs

看这篇文章需要具备的知识:

  • nodejs
  • npm
  • git

如果以上都了解就可以在github上免费搭建一个自己的博客了

准备工作:

  • 电脑安装好nodejs
  • 电脑安装好git
  • github帐号

1. 安装hexo

在终端中输入

npm install -g hexo-cli

等待安装完成后就可以使用hexo了

在终端中找到自己喜爱的文件夹位置输入

 hexo init myBlog

myBlog也可以替换成自己喜欢的名字

过一会博客文件创建好了，继续输入

cd myBlog
npm install
npm install hexo-deployer-git --save

等待安装完成就可以进行下一步了

2.配置hexo

在配置hexo前需要在github创建一个repo

Repository name输入

你喜欢的名字.github.io

例如

myshiningblog.github.io

创建好后复制下来git地址 例如

git@github.com:yourname/myshiningblog.github.io.git

进入settings找到 Deploy keys

点击

Add deploy key

把你电脑里的ssh-key复制进去

记得允许写入权限

回到你创建好的的博客目录
在博客目录下找到_config.yml并打开

修改网站的基本信息

title: 网站的标题
subtitle: 副标题
description: 描述
keywords: 关键词
author: 作者
language: zh-CN

在最下方找到

deploy:
  type: git
  repo: <repository url>
  branch: [branch]

修改成

deploy:
  type: git
  repo: 复制下来的git地址
  branch: master

保存！

3.生成和发布

在终端输入

hexo g -d

等一会就发布成功了

打开github.io的页面

你喜欢的名字.github.io

就能看到你的博客啦

4.创作编辑文章

有两种方法可以创建编辑文章

1.使用代码编辑器 例如 sublime vscode atom

用终端进入博客目录下 输入

hexo new 文章的标题

这时候在博客的目录

/source/_posts/

下就能看到你新创建的文章的标题.md了

直接就可以在这里写你的文章了

2.使用本地服务后台

用终端进入博客目录下 输入

npm install --save hexo-admin

安装完成后 输入

hexo server

出现

Hexo is running at http://localhost:4000/

就说明服务启好了
访问这个地址就能在本地查看博客

访问
http://localhost:4000/admin

进入后台就可以对文章进行创建编辑了

文章写完后记得在终端博客目录下 输入

hexo g -d

这样文章就会生成并发布到github上了

5.博客页面修改

在博客目录下/themes/里就是主题文件了

有一个 landscape 的默认主题

如果对自己html css有信心，并且熟悉ejs模板还有stylus

就可以从默认模板下手修改成自己喜欢的博客样式

如果自己没有太强的前端知识
可以访问
https://hexo.io/themes/

挑选自己喜欢的模板并按照 主题中的readme操作就可以更换模板了
